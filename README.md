# To-Do List (React)

Простое To-Do List приложение на React.  
Позволяет добавлять задачи, редактировать, удалять и отмечать их как выполненные.  

## Функционал
- Добавление новых задач  
- Удаление задач  
- Редактирование существующих задач  
- Отметка задачи как выполненной / отмена выполнения  

## Установка и запуск

1. Клонировать репозиторий:
   ```bash
   git clone https://github.com/vedko04/React-to-do-list
   cd React-to-do-list
Установить зависимости:
```bash
npm i
```
Запустить проект:
```bash
npm start
```
